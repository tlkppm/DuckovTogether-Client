# DuckovTogether Client

A multiplayer modification client for cooperative gaming experiences.

多人联机模组客户端，提供合作游戏体验。

## Overview | 概述

DuckovTogether Client is an **independently developed** multiplayer modification. This project is created from scratch and is **not affiliated with, derived from, or based on any other project or codebase**.

DuckovTogether Client 是一个**独立开发**的多人联机模组。本项目从零开始创建，**与任何其他项目或代码库无关联、无衍生关系**。

This software is developed independently by tlkppm and contributors.

本软件由 tlkppm 及贡献者独立开发。

## Features | 功能特性

- **Multiplayer Support | 多人联机** - Play cooperatively with friends | 与好友合作游戏
- **Player Synchronization | 玩家同步** - Real-time position and state synchronization | 实时位置和状态同步
- **AI Synchronization | AI同步** - Synchronized AI behavior across all players | 所有玩家间同步AI行为
- **Item Synchronization | 物品同步** - Shared loot and inventory synchronization | 共享战利品和物品栏同步
- **Dedicated Server Support | 专用服务器支持** - Connect to DuckovTogether Server | 连接到 DuckovTogether 服务器

## Installation | 安装

1. Download the latest release | 下载最新版本
2. Extract to game's `Mods` folder | 解压到游戏的 `Mods` 文件夹
3. Launch the game | 启动游戏

## Usage | 使用方法

1. Press the configured hotkey to open the mod menu | 按配置的快捷键打开模组菜单
2. Enter server IP and port | 输入服务器 IP 和端口
3. Connect and play | 连接并游玩

## Building from Source | 从源码构建

```bash
git clone https://github.com/tlkppm/DuckovTogether-Client.git
cd DuckovTogether-Client
dotnet build -c Release
```
